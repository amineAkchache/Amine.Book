About BookTracker

Track Your Reading Journey

BookTracker is a web application designed to help book lovers organize their reading life. Whether you're an avid reader who goes through dozens of books a year or someone who just wants to build a more consistent reading habit, BookTracker provides the tools you need.

Key Features

Book Search

Find books using the Google Books API or add books manually to your library.

Reading Progress

Keep track of which books you want to read, are currently reading, or have finished.

Reading Challenges

Set goals to read a certain number of books within a timeframe.

Reviews & Ratings

Keep notes on books you've read and rate them on a 5-star scale.

Technology Stack

BookTracker is built using the following technologies:

  • Backend: Flask, Python, SQLAlchemy
  • Database: MySQL
  • Frontend: Bootstrap 5, HTML5, CSS3, JavaScript
  • API: Google Books API
  • Authentication: Flask-Login, bcrypt for password hashing
